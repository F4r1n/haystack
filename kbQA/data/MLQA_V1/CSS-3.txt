CSS-3 ist eine zweistufige Flüssigtreibstoff-Rakete. Als Treibstoff verwendet die CSS-3 die lagerfähigen Flüssigtreibstoffe UDMH und Stickstofftetroxid. Die Steuerung erfolgt mittels einer Trägheitsnavigationsplattform. Sie hat einen Startschub von 1.224 kN, eine Startmasse von 82.000 kg, einen Durchmesser von 2,25 m, eine Länge von 28,05 m und eine Flossenspannweite von 2,74 m. Die Reichweite der CSS-3, die mit einem Atomsprengkopf von 3.300 kt Sprengkraft und einem Gewicht von 2.190 kg bestückt ist, beträgt 4.750 km. Es wird eine Präzision (CEP) von 1.500 m erreicht.
